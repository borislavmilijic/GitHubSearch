# GitHubSearch
Simple application which can be used to search for repositories on Github.

### Requirements:

- [x] Use Kotlin
- [x] Input Text Search Field
- [x] List for Results
- [x] Use Github REST API
- [ ] Implement pagination
- [ ] Result entries clickable -> Open in system browser
